# Online Retail Sales Analysis using Excel

## Project Overview
This project analyzes an online retail dataset using Microsoft Excel. The project covers the complete data analysis workflow from data auditing and cleaning to business analysis using Pivot Tables.

## Dataset
- Source: Online Retail Dataset
- Rows: 541,910
- Columns: 8

## Tools Used
- Microsoft Excel
- Pivot Tables
- Excel Functions

## Project Workflow
1. Data Audit
2. Data Cleaning
3. Data Preparation
4. KPI Analysis
5. Business Analysis
6. Insights & Recommendations

## KPIs
- Total Revenue
- Total Orders
- Total Customers
- Total Quantity Sold
- Average Order Value

## Business Questions
- What are the top products by revenue?
- What are the top products by quantity?
- Which month generated the highest revenue?
- Which country generated the highest revenue?
- Who are the top customers?
- Which products are returned the most?

## Key Insights
- "DOTCOM POSTAGE" generated the highest revenue.
- "WORLD WAR 2 GLIDERS ASSTD DESIGNS" was the best-selling product.
- November generated the highest revenue.
- The United Kingdom generated the highest revenue.
- Customer 14646 was the highest spending customer.
- "Printing smudges/thrown away" was the most returned product.

## Recommendations
- Increase inventory for top-selling products.
- Launch marketing campaigns before November.
- Retain high-value customers with loyalty programs.
- Investigate the causes of frequent product returns.
- Continue investing in the UK market while expanding internationally.
